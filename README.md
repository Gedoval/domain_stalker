# Domain stalker

## Overview

Interactive DNS tool, built with Rust and modularity in mind.

## Setup (Release)

Release ready version.

```
$ PLUGINS_DIRECTORY=plugins ./domain_stalker
```

## Setup (Developer)

Developer version.

```
$ cargo run
```
